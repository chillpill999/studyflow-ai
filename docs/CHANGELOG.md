# Changelog

All notable changes to **The Study Flow** will be documented in this file.

## [1.0.0] - 2026-06-29

### Added
- **UI/UX & Themes**: Pearl Glass, Dark, and Light mode settings persisted in browser storage.
- **Top Navigation Bar**: Mounts Command Palette search, Theme toggles, and Notification dropdown lists.
- **Command Palette**: Triggered by `Ctrl + K` with full keyboard arrow controls and ESC handling.
- **Accessibility**: Automatic fallback for `prefers-reduced-motion` disabling high-intensity canvas renders.
- **Error Monitoring**: Intercepts unhandled errors/rejections, converting them directly to toast notification cards.
- **Production Guides**: Added architectural README, API, DEPLOYMENT, and SECURITY guidelines.
